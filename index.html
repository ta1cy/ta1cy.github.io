1st Place - UCSB Hackathon

IoT Plant Observation System

ArduinoC++PythonFlaskOpenCV

Overview

  • This project was developed as a submission for the UCSB 10th Hackathon event.
  • It consists of an Arduino-based IoT system created to monitor plant health in real time.
  • The platform uses soil moisture, humidity, temperature sensors to track environmental conditions around the plant.
  • A live webcam feed is processed through several image preprocessing stages to detect leaf discoloration and possible anomalies.
  • All sensor readings and camera analysis results are streamed to a Flask server, where a dashboard displays real time data, visualizations, and time series logs.

My Contributions

  • Sensor acquisition board and integrated the and temperature sensors.
  • Computer vision module, which includes grayscale conversion, thresholding, contrast adjustments, and an anomaly classifier fordetecting leaf discoloration.

Results

  • The system was able to monitor the various plant environments and generate data.
  • This project won first place in the Environmental category of the UCSB 10th Hackathon event.
  • You can read a demonstration on the full project description and see the Devpost page Thirsty Plants.

SpringMorph Haptic User Study

3D PrintingEmbedded SystemsUnitySerial Comms

Overview

  • This project was part of my work at the Human AI Experience Lab, supporting the SpringMorph haptic user study led by visiting PhD student Carlos Gilberto.
  • It consists of an 18 unit array of spring-powered actuators designed to generate programmable shape shifting tactile feedback for VR and MR environments.
  • The system uses 3D printed mechanical components, embedded sensors, and actuators to produce precise physical servo deformation patterns for user studies in haptic perception.

My Contributions

  • Led the precision 3D printing, mechanical assembly workflow, and debugging process for the platform, including fabrication and calibrationof all sensors and servo motion.
  • Helped design and implement the serial communication protocol that coordinated individual and group actuator behavior.
  • Assisted with integration between Unity and actuator array by establishing a bidirectional serial pipeline and building example VR and MR environments using the Meta XR SDK.

Results

  • Produced a fully assembled and calibrated actuator array suitable for lab demonstrations and early user study testing.
  • Supported the preparation of the SpringMorph system for demonstration and user study development within the HAX Lab.

Online News Popularity Classifier

PythonK-MeansSVDNeural Networks

Overview

  • This project was developed as the final course project for EC ENGR 133A, where we analyzed the Online News Popularity dataset containing 39,644news articles and 58 features.
  • The goal was to build a classification system that predicts whether an article will be "popular" based on textual, metadata, and engagement-related features.

My Contributions

  • Implemented all preprocessing steps, including standardization, feature reduction, correlation filtering, and dataset transformations (log, square, sqrt).
  • Developed and evaluated the linear classification pipeline, including K-means clustering, SVD analysis, and FLOP/numerical-stability calculations.
  • Built the nonlinear MLP classifier that achieved near-perfect accuracy and AUC, and performed full training/testing evaluation with confusion matrices.

Results

  • Linear models achieved around 55–63%, depending on engineering.
  • Adding nonlinear transformations accuracy drastically to around 87% improved (AUC ≈ 0.955).
  • The final neural-network classifier achieved near-ideal performance: ~99.2% accuracy, 0.999 AUC, and very low RMS error. Demonstrating that the nonlinear model demonstrated the best numerical stability and generalization.

Smart Portrait Framing

PythonYOLOv8MediaPipeComputer Vision

Overview

  • This project was developed as a standalone computer vision system for dynamic portrait framing and multi-person tracking, inspired by my applied vision work during my internship experience at DTEN.
  • It consists of a real-time face detection, identity tracking, and layout generation pipeline built on top of YOLO models, face-mesh tracking, and stabilization algorithms.
  • The system uses two primary vision modules: a face detector for global framing and a dense 478-point face mesh model for fine-grained head orientation and speaker-aware focus.
  • A smoothing and re-entry correction mechanism is applied to reduce jitter, remove duplicates, and ensure consistent identity assignment across frames.

My Contributions

  • Implemented the complete framing system in Python, including face detection, mesh tracking, and bounding box fusion using YOLOv8 face and Google MediaPipe Face Landmarker.
  • Designed the identity stabilization module that uses temporal filtering, head pose estimation, and de duplication rules to maintain stable identity assignments across multiple faces.

Results

  • Produced a fully functional real time portrait framing demo capable of tracking multiple subjects and generating stable cropped outputs.
  • Full implementedtation, documentation, and code are available on the GitHub project smartframing.
